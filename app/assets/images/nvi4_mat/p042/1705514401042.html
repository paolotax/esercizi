Il comportamento dello zero e dell’uno nella divisione


0 : 15 = 0
0 : 257 = 0

0 diviso qualsiasi numero dà 0.



15 : 0 = impossibile

Non è possibile dividere un numero per 0, perché non esiste un numero che moltiplicato per 0 dia come risultato il numero stesso.

 

 

58 : 1 = 58
137 : 1 = 137

Se dividi un numero per 1 hai come risultato sempre il numero stesso.

 

 

58 : 58 = 1
137 : 137 = 1

Se dividi un numero diverso da zero per se stesso hai come risultato sempre 1.
