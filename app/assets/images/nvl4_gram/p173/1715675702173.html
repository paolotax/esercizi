Il modo indicativo

 

1 IMPARARE TUTTI

Sottolinea i verbi. Poi indica con una X se si tratta di un tempo semplice (S) o di un tempo composto (C).

 

• Martina è andata dal dentista.       S C
• Il treno è partito alle nove.       S C
• Avevo dimenticato i soldi.       S C
• Ascoltava con molta attenzione.       S C
• Sono orgoglioso di te.       S C
• Soffiava un forte vento.       S C
• Bravissimo! Hai indovinato!       S C
• Domani andrò in gita scolastica.       S C
• Era ormai sera.       S C
• Avevate fatto un bel lavoro.       S C

 

 

2 In ogni frase coniuga il verbo tra parentesi al tempo passato adatto.
Osserva l’esempio.

  • Mentre (leggere) leggeva un libro, qualcuno entrò in casa sua.
  • Ieri io (mangiare) ................... una buonissima pesca appena colta.
  • In passato gli uomini (coltivare) ................... la terra senza l’aiuto delle macchine.
  • Cesare (invadere) ................... le Gallie.
  • Dopo che ebbe controllato la situazione dell’auto, (partire) ................... tranquillo.

 

 

3 Completa coniugando il verbo tra parentesi a un tempo semplice del modo indicativo, poi scrivi quale tempo hai usato.

 

• Io e Martina (camminare) ................... per mano. → ......................................
• Voi (parlare) ................... troppo. → ......................................
• Domani noi (andare) ................... a teatro. → ......................................
• Questo libro ti (aiutare) ................... a capire. → ......................................
• Lisa (cadere) ................... mentre sciava. → ......................................

 

 

4 Riscrivi le frasi trasformando i tempi semplici in tempi composti. Osserva l’esempio.
