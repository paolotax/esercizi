Analizzare una frase

 

• Per fare l’analisi grammaticale di una frase puoi seguire questi passaggi.

1 Prima di tutto, dividi la frase in parole.

 

La nonna abbraccia dolcemente il nipote → La / nonna /  abbraccia / dolcemente / il / nipotino.

 

 

2 Per ogni parola, domandati: è una parte del discorso variabile (cambia la desinenza) o invariabile (non cambia la desinenza)?

 

La può cambiare desinenza?       Sì No       quindi è variabile
nonna può cambiare desinenza?       Sì No       quindi è variabile
abbraccia può cambiare desinenza?       Sì No       quindi è variabile
dolcemente può cambiare desinenza?       Sì No       quindi è invariabile
il può cambiare desinenza?       Sì No       quindi è variabile
nipotino può cambiare desinenza?       Sì No       quindi è variabile

 

 

3 Se è una parte variabile, domandati:

  • indica un modo di essere, uno stato o un’azione?
    Se la risposta è sì, allora è un verbo.
  • è una persona, un animale o una cosa?
    Se la risposta è sì, allora è un nome.
  • accompagna un nome?
    Se la risposta è sì, allora può essere un articolo o un aggettivo.
  • sostituisce un nome?
    Se la risposta è sì, allora è un pronome.

 

 

4 Se è una parte invariabile, domandati se si tratta di: una preposizione, un avverbio, una congiunzione, un’esclamazione. Per ricordare, osserva lo schema.
