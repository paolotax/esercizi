2 In ogni coppia di frasi indica con una X quella suddivisa correttamente in sintagmi.
Poi sottolinea il verbo e cerchia il complemento oggetto della frase che hai scelto.
Osserva l’esempio.

 

• Il bagnino chiude / gli ombrelloni.        
• Il bagnino / chiude /       X
• Metti / l’astuccio / nello zaino.        
• Metti / l’astuccio nello / zaino.        
• La strega / trasformò / il principe in / un ranocchio.        
• La strega / trasformò / il principe / in un ranocchio.        
• I nonni / mi / hanno regalato / un nuovo libro.        
• I nonni mi / hanno regalato / un nuovo / libro.        
• Il papà / ha cotto / le pizze / nel forno / a legna.        
• Il papà / ha cotto / le pizze / nel forno a legna.        
• Io indosso / quasi sempre / i jeans.        
• Io / indosso / quasi sempre / i jeans.
